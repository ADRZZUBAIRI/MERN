List Types

There are two types of lists in HTML:

  • Unordered List
  • Ordered List

Unordered List

Unordered lists are lists that have a bullet point or some other mark before the text.

  • Coffee
  • Tea
  • Milk
  • Water

Ordered List

Ordered lists are lists that have a number or some other mark before the text.

  1. Coffee
  2. Tea
  3. Milk
  4. Water

Description List

Description lists are lists of terms with a description of each term.

Coffee
- black hot drink
Tea
- green hot drink
Milk
- white cold drink
Water
- transparent cold drink
